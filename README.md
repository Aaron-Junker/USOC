# How can I help
You can read our [Contibuting Document](https://github.com/Case-Games/USOC/blob/master/CONTRIBUTING.md) or our [Code of Conduct](https://github.com/Case-Games/USOC/blob/master/CODE_OF_CONDUCT.md)
